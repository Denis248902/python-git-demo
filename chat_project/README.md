# Асинхронный чат-сервер (asyncio)

Простой TCP чат на asyncio: поддержка нескольких клиентов, бродкаст сообщений, уведомления о входе/выходе, команды /nick, /help.

## Запуск

1. Сервер: `python server.py`
2. Клиенты: `python client.py` (запустить в нескольких терминалах)

## Порты

Сервер слушает порт 8888.
