# Linear Search in Python

Реализация линейного поиска, тесты и замеры производительности.

## Что реализовано

- Функция `linear_search`: возвращает индекс элемента или -1.
- Тесты на списке из 100 случайных чисел.
- Замеры времени для разных размеров списка.
- График зависимости времени от размера (если установлен matplotlib).

## Как запустить

1. Установите зависимости (опционально, для графика):
   \`\`\`bash
   pip install matplotlib
   \`\`\`
2. Запустите скрипт:
   \`\`\`bash
   python linear_search.py
   \`\`\`

## Сложность

- Временная сложность: O(n) в худшем и среднем случаях, O(1) в лучшем.
- Пространственная сложность: O(1).
