# Проект по курсу «Основы синтаксиса языка программирования Python»

## Краткое описание
Учебный проект, выполненный в рамках курса. Демонстрирует освоение базовых конструкций языка Python.

## Реализованные задачи
- Простой калькулятор
- Проверка чётности числа
- Подсчёт статистики по списку чисел
- Конвертер температур (Цельсий ↔ Фаренгейт)
- Игра «Угадай число»

## Структура проекта
```
python-basics/
├── calculator.py
├── even_odd.py
├── stats.py
├── converter.py
└── guess_game.py
```

## Как запустить
1. Убедитесь, что установлен Python 3.6+
2. Выполните: `python имя_файла.py` (например, `python calculator.py`)
